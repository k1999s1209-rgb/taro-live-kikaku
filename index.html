カラオケ5番勝負

精密採点Aiで5本先取。点差が開くほど、罰が重い。

日程：8/7（金）あたり 仮 場所：カラオケ店（店舗選定中） 尺：約3時間 生配信
まずこれだけ

ひろまるさん、8/7（金）前後の都合を教えてください

店舗（精密採点Aiが使えるカラオケ店）はたろが選定中。日程が固まったら店の予約と合わせて確定します。罰ゲームと妨害ギフトは7/21に確定しました（下記）。

企画概要

  • たろ vs ひろまるが1曲ずつ歌い、DAM精密採点Aiの点数（小数第3位まで）で勝負。先に5勝した方が総合勝者（5-0の最短5ラウンド〜4-4からの最長9ラウンド。消化試合ゼロで、もつれるほど1曲がマッチポイントになる）
  • ラウンドごとに敗者は罰を即実行——しかも点差が開くほど罰が重くなる3段階制。総合敗者には最後にデカ罰
  • 視聴者はギフトで妨害・応援・お題決定ができ、点数ピッタリ予想が当たると賞金（コメントだけで参加無料）
  • ※歌うま配信ではありません。素人2人がAI採点に裁かれる配信です——点数がバラつくほど予想も罰も盛り上がります

ルール（1ラウンド約20分の流れ）

  1. お題タイム：ジャンルをお題ルーレットで決定。「大好き」ギフトを最初に投げた人がいれば、その人がジャンルを指定できる
  2. 予想タイム → たろ歌唱 → 予想タイム → ひろまる歌唱
  3. 結果発表：勝敗＋点差（＝罰の重さ）＋ピッタリ賞の当落が一気に決まる、1番の山場
  4. 敗者は点差レベルの罰（下の表）を即実行 → 次のラウンドへ
  5. 先に5勝で決着 → 総合敗者にデカ罰：ケツバット3連発＋視聴者投票で追加1品

星取表は常時掲示（途中参加でも「いま3-2で王手」が一目で分かる）。曲の使い回しは禁止。長引いたら6ラウンド目以降はワンハーフ歌唱に切替、早く決着したら視聴者リクエスト歌唱のエキシビションで締めます。

点差＝罰の重さ（3段階）

点差レベル罰（7/21確定）
0.999以下僅差 → 軽罰激辛の簡単なものを食べる
1.000〜2.999中罰混ぜドリンク
3.000以上大差 → 激罰ゴムパッチン

点数が読み上げられる瞬間に「激罰キター！」まで決まる仕組み。閾値は初回の実測点差でチューニングします。

罰メニュー（7/21確定・レベル別固定）

  1. 軽罰激辛の簡単なものを食べる（激辛スナック・激辛ソースひと舐め など）
  2. 中罰混ぜドリンク（ドリンクバーで数種を混ぜた一杯を飲み干す）
  3. 激罰ゴムパッチン

くじ方式はやめてレベル別の固定罰に。ケツバットは総合デカ罰にのみ登場します。

視聴者参加の4本柱

1. お題（ジャンル）決定＝ルーレット基本＋大好き（500コイン）で指定権【7/21確定】

各ラウンド冒頭、ジャンルはお題ルーレットで決定（90年代・アニソン・夏うた等のテーマ指定が基本）。「大好き」ギフトを最初に投げた人がいれば、その人がジャンルを指定できる。

2. 妨害ギフト【7/21確定・6種】

ギフトごとに妨害カード発動。どちらを妨害するかは視聴者が指定。1曲につき最大2回まで＝点数勝負は壊さない。

  1. ハンドハートキー変更（100コイン）
  2. ハンドポーズマイク割り込み3秒（100コイン）
  3. マネーガンハリセン
  4. コーギーピコピコハンマー
  5. ドーナツチキン
  6. 折り鶴倍速10秒

3. 応援加点ギフト

各歌唱直前の「応援タイム30秒」のギフト量に応じて +0.2〜最大+1.0点を加点。僅差はひっくり返るが、大差は覆らない上限設定。

4. ピッタリ賞（点数予想で賞金・参加無料）

歌唱前に「86.534」形式の点数フル予想をGoogleフォームで送信（歌い出しで締切・1人1回・送信時刻の早い方が先着）。参加は完全無料、各賞先着1名。結果発表のタイミング（採点のたび or 最後にまとめて）は調整中。

ちょっと福岡行ってきました。— まだ見ぬ福岡に、会いに行く。

日程：未定（構想段階） 範囲：福岡県 全72エリア 形式：検討中（ライブ or 動画）

どんな企画？

  • 福岡県を72エリア（北九州市7区＋福岡市7区＋58市町村）に分け、1エリアずつ訪ねて全制覇を目指す長期シリーズ
  • その町で頑張っている若者・お店・企業を応援し、地元グルメと町の魅力を紹介する
  • 締めのくじで次に行く町が決まる（エリア番号を視聴者がランダムに選ぶ）→ 出た町のおすすめをその場で聞く
  • 地域を盛り上げることは、たろの起業テーマ（地域再興）そのもの。本気でやる企画

目的（5つ）

  1. その町で頑張っている若者・店舗・企業の応援
  2. 地元のグルメ紹介
  3. 地元の魅力紹介
  4. 紹介を通じた企業とのタイアップなど、仕事への連携（ひろまるさん・たろの仕事につながる）
  5. シリーズ化による視聴者数の増加

ルール（5つ）

  1. 最低限の下調べをして行く
  2. 一度行った地域には行かない（全エリアを回るため）
  3. ギフトは「移動費・現地の食費・応援に使う資金」として、使い道を公表して扱う
  4. 動画の最後に、次に行く町をくじ（エリア番号）で視聴者にランダムに選んでもらい、出た町のおすすめを聞く
  5. 視聴者はスポンサーではなく協力者・参加者という位置付け

エリアの分け方（全72）

  • 1〜7：北九州市の7区（門司・小倉北・小倉南・若松・八幡東・八幡西・戸畑）
