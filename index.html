かえれま10

人気メニューTOP10、全部当てるまで帰れません。

日程：8/1（土）仮 → 平日になる可能性あり 場所：はま寿司（許可打診中）／マックも候補 尺：90〜120分 生配信
まずこれだけ

7月最終週〜8月上旬で「動ける平日」の候補日をください

土曜だとお店の撮影許可が難しい可能性があります。許可の返答が来たらすぐ日程を確定したいので、平日候補を先にけいたろへ送ってください。

企画概要

  • けいたろ＆ひろまるさんが店舗でメニューを注文。人気TOP10に入っていれば ○、外れたら ×
  • 対象は全メニュー（寿司ネタ・サイド・デザート・ドリンク全部）
  • TOP10を全て引き当てたらクリア＝帰れる。ハズレでも必ず完食・完飲
  • 答えを知っているのはむらさこさんだけ。挑戦する2人も視聴者も知らない → ネタバレなしで、視聴者もコメントで一緒に予想できる

役割分担

挑戦者
けいたろ
注文・完食・配信の進行
挑戦者
ひろまるさん
注文・完食・カメラとコメント拾いを分担
出題・判定人
むらさこさん
答え（TOP10リスト）を一人だけ持ち、注文ごとに○×を判定。遠隔参加でOK

ルール（ゲームの流れ）

  1. 注文は1回1品ずつ宣言する
  2. むらさこさんがLINEで ○ か × だけ返す（品名のヒントは無し）
  3. ハズレも必ず食べ切る／飲み切ってから次へ（2人で分けるのはOK）
  4. TOP10全部当てたらクリア＝帰れる
  5. 制限時間90分 or 注文30品で終了 → 正解TOP10を発表して答え合わせ
    罰ゲームは無し。答え合わせが締めの山場

ヒント制度（ギフト連動・最大3回）

視聴者からのTikTokギフトが条件を満たすとヒント解放。出すのはむらさこさん。

  1. 1回目ジャンルヒント（「未正解ににぎり系が△品ある」）
  2. 2回目頭文字ヒント（「未正解のうち1品の頭文字は『◯』」）
  3. 3回目二択ヒント（「候補に挙がった2品のうち入っているのは片方だけ」）

ギフトが少なくても「ハズレ5連続で救済ヒント1回」あり。

お二人へのお願い

むらさこさん

  1. ネットの人気ランキングサイトを1つ選び、そのTOP10を正解リストにする（全メニュー型で。どのサイトかは答え合わせまで秘密）
  2. 配信中は「○」「×」だけ返す
  3. ヒント文面は事前にすり合わせ
  4. 答えは配信終了まで誰にも漏らさない（この企画の生命線）

ひろまるさん

  1. 平日の候補日を共有（上の「まずこれだけ」）
  2. 当日の分担相談：完食のシェア、カメラ・コメント拾い
  3. 当日は空腹で来る（ハズレが続くと満腹地獄です）

予算

ハズレ込みで20〜30品、4,500〜7,500円を想定（上限1万円まで）。費用の分担は別途相談。

全員で守ること（炎上防止）

  1. お店の撮影許可の条件に必ず従う（許可なしでは配信しない）
  2. 他のお客様・店員さんは絶対に映さない（カメラは自分たちのテーブルだけ）
  3. 食べ残しゼロ・大声などの迷惑行為なし
  4. 配信中にお店へのネガティブ発言をしない（宣伝になる内容として許可をもらう前提）

カラオケ10本勝負

精密採点Aiでガチ勝負。負けたら即、罰くじ。

日程：8月中〜下旬 予定（調整中） 場所：配信OKのカラオケ店を選定中 尺：2.5〜3時間 生配信
まずこれだけ

ひろまるさん、8月中〜下旬で動ける候補日をください

場所（配信OK×精密採点Aiのカラオケ店）はけいたろが選定中。候補日が揃ったら店の予約と合わせて日程を確定します。

企画概要

  • たろ vs ひろまるが1曲ずつ歌い、DAM精密採点Aiの点数（小数第3位まで）で勝負 × 全10ラウンド
  • ラウンドごとに敗者は「罰くじ」を引いて即実行。10戦の星取で負け越した方には最後にデカ罰
  • 視聴者はギフトで妨害・応援ができ、点数ピッタリ予想が当たると賞金（コメントだけで参加無料）
  • 10〜15分ごとに勝敗＋罰ゲームの山場が来るので、途中から見ても楽しめる設計。月1のレギュラー企画にしたい

ルール（1ラウンドの流れ）

  1. 視聴者の予想タイム → たろ歌唱 → 予想タイム → ひろまる歌唱
  2. 精密採点Aiの総合点が高い方が勝ち（結果発表でピッタリ賞も判定）
  3. 敗者は罰くじを引いて即実行（1ラウンド 約13〜15分）
  4. これを10ラウンド。星取表を常時掲示するので途中参加でも戦況が一目で分かる
  5. 総合敗者にはデカ罰：ケツバット3連発＋視聴者投票で追加1品

選曲は同一配信内で使い回し禁止。選曲方式（お題縛りなど）は2人で相談して決める。

視聴者参加の3本柱

1. 妨害ギフト

指定ギフトで妨害カード発動（キー±2変更・エコーOFF・被り物・変顔札など）。どちらを妨害するかは視聴者が指定。1曲につき最大2回まで＝点数勝負は壊さない。

2. 応援加点ギフト

各歌唱直前の「応援タイム30秒」のギフト量に応じて +0.2〜最大+1.0点を加点。僅差はひっくり返るが、大差は覆らない上限設定。

3. ピッタリ賞（点数予想で賞金・参加無料）

歌唱前に「86.534」形式で点数をフル予想してコメント（歌い出しで締切・1人1回）。参加は完全無料、各賞先着1名。

一致例（正解 86.534 の場合）賞金
下3桁一致◯◯.5341,000円
下4桁一致◯6.5345,000円
完全一致86.53410,000円

支払いはAmazonギフトコード等をDMで送付。1配信の賞金総額は上限3万円（到達したらその回は終了）。

罰くじ（10枚）

  • ケツバット1発 ×3枚（定番枠。おもちゃのカラーバット使用）
  • 顔にシール1枚追加（配信終了まで剥がせない）／被り物・変装で次の曲を歌う
  • 次ラウンドは相手選曲／語尾縛り（「〜にゃん」等）
  • 視聴者お題のモノマネ／全力エア熱唱／視聴者おまかせ（コメント多数決）

ひろまるさんへのお願い

ひろまるさん

  1. 8月中〜下旬の候補日を共有（上の「まずこれだけ」）
  2. 選曲ルールの相談（お題縛り自選が有力案）
  3. 費用とギフト収益の分配ルールを事前にすり合わせ
  4. 当日は喉を仕上げてくる（10曲歌います）

この企画の体制

  1. たろ vs ひろまるの2人企画（むらさこさんの担当役はなし。観戦・参戦アイデア歓迎）
  2. ピッタリ賞の判定は2人で兼任（締切時にコメント欄をスクショ→結果発表時に照合）

全員で守ること

  1. ピッタリ賞は参加無料を徹底（ギフトを参加条件にしない。「コメントだけで誰でも参加無料」を毎回明言）
  2. ケツバットはおもちゃバット＋軽く当ててリアクションで盛るバラエティ演出に統一（ガチの痛がり方はNG）
  3. お店の配信許可の条件に必ず従う
  4. 入店後の音量リハを必ずやる（マイクとミュージックのバランス）
