かえれま10

人気メニューTOP10、全部当てるまで帰れません。

日程：8/1（土）仮 → 平日になる可能性あり 場所：はま寿司（許可打診中）／マックも候補 尺：約2時間 生配信（店舗に着いてからスタート）
まずこれだけ

7月最終週〜8月上旬で「動ける平日」の候補日をください

土曜だとお店の撮影許可が難しい可能性があります。許可の返答が来たらすぐ日程を確定したいので、平日候補を先にたろへ送ってください。

企画概要

  • たろ＆ひろまるさんが店舗でメニューを注文。人気TOP10に入っていれば ○、外れたら ×
  • 対象は全メニュー（寿司ネタ・サイド・デザート・ドリンク全部）
  • TOP10を全て引き当てたらクリア＝帰れる。ハズレでも必ず完食・完飲
  • 答えを知っているのはむらさこさんだけ。挑戦する2人も視聴者も知らない → ネタバレなしで、視聴者もコメントで一緒に予想できる

役割分担

挑戦者
たろ
注文・完食・配信の進行
挑戦者
ひろまるさん
たろと対等な演者として出演。注文・完食・カメラとコメント拾いを分担（画角も2人が等しく映る形が基本）
出題・判定人
むらさこさん
答え（TOP10リスト）を一人だけ持ち、注文ごとに○×を判定。遠隔参加でOK

ルール（ゲームの流れ）

  1. 注文は5品ずつまとめて宣言する（寿司なら5皿・マックなら5品を1セット。毎回1品ずつだとお店の負担になるため）
  2. むらさこさんが配信のコメント欄に ○ か × だけ書き込む（品名のヒントは無し）。正解のときだけ「○！3位」のように何位かも発表
  3. ハズレも必ず食べ切る／飲み切ってから次の5品へ（2人で分けるのはOK・上限30品=最大6セット）
  4. TOP10全部当てたらクリア＝帰れる
  5. 制限時間90分 or 注文30品で終了 → 正解TOP10を発表して答え合わせ
    罰ゲームは無し。答え合わせが締めの山場

視聴者参加ギフト（コイン別）

ギフトのコイン数で「できること」が変わります。各ヒントは1配信につき1回ずつ。出すのはむらさこさん（判定と同じコメント欄で）。

  1. 50ネタリクエスト権「このネタを食べてほしい」→ 2人は次の5品セットに必ず入れて食べる
  2. 100ジャンルヒント（「未正解ににぎり系が△品ある」）
  3. 500頭文字ヒント（「未正解のうち1品の頭文字は『◯』」）
  4. 10004択ヒント（「この4品のうち1つだけがTOP10入り」）

さらに、ハズレが5連続したら配信から「お楽しみ袋」（視聴者へのコイン山分け）を投げるお詫びイベントあり。

お二人へのお願い

むらさこさん

  1. ネットの人気ランキングサイトを1つ選び、そのTOP10を正解リストにする（全メニュー型で。どのサイトかは答え合わせまで秘密）
  2. 配信中はコメント欄で「○」「×」だけ書き込む（正解のときは順位も発表。見やすいようモデレーター登録します）
  3. ヒント文面は事前にすり合わせ
  4. 答えは配信終了まで誰にも漏らさない（この企画の生命線）

ひろまるさん

  1. 平日の候補日を共有（上の「まずこれだけ」）
  2. 当日の分担相談：完食のシェア、カメラ・コメント拾い
  3. 当日は空腹で来る（ハズレが続くと満腹地獄です）

予算

総予算は約2万円（飲食4,500〜7,500円＋お楽しみ袋の原資＋機材・予備など込み）。費用の分担は別途相談。

打ち合わせで決めること（次回MTGの議題）

  • お楽しみ袋の金額・当選人数
  • 各コイン数をどのギフト（名前・見た目）にするか
  • リクエスト殺到時のルール（お一人様1回まで など）

  • 当日の分担（判定リハ・下見・切り抜き）

全員で守ること（炎上防止）

  1. お店の撮影許可の条件に必ず従う（許可なしでは配信しない）
  2. 他のお客様・店員さんは絶対に映さない（カメラは自分たちのテーブルだけ）
  3. 食べ残しゼロ・大声などの迷惑行為なし
  4. 配信中にお店へのネガティブ発言をしない（宣伝になる内容として許可をもらう前提）

カラオケ5番勝負

精密採点Aiで5本先取。点差が開くほど、罰が重い。

日程：8/7（金）あたり 仮 場所：カラオケ店（店舗選定中） 尺：約3時間 生配信
まずこれだけ

ひろまるさん、8/7（金）前後の都合を教えてください

店舗（精密採点Aiが使えるカラオケ店）はたろが選定中。日程が固まったら店の予約と合わせて確定します。罰くじの中身も一緒にブラッシュアップしたいです。

企画概要

  • たろ vs ひろまるが1曲ずつ歌い、DAM精密採点Aiの点数（小数第3位まで）で勝負。先に5勝した方が総合勝者（5-0の最短5ラウンド〜4-4からの最長9ラウンド。消化試合ゼロで、もつれるほど1曲がマッチポイントになる）
  • ラウンドごとに敗者は罰くじを引いて即実行——しかも点差が開くほど罰が重くなる3段階制。総合敗者には最後にデカ罰
  • 視聴者はギフトで妨害・応援・お題決定ができ、点数ピッタリ予想が当たると賞金（コメントだけで参加無料）
  • ※歌うま配信ではありません。素人2人がAI採点に裁かれる配信です——点数がバラつくほど予想も罰も盛り上がります

ルール（1ラウンド約20分の流れ）

  1. お題タイム：指定ギフトを最初に投げた視聴者がそのラウンドの選曲テーマを決定（ギフトが無ければお題ルーレットを回す）
  2. 予想タイム → たろ歌唱 → 予想タイム → ひろまる歌唱
  3. 結果発表：勝敗＋点差（＝罰の重さ）＋ピッタリ賞の当落が一気に決まる、1番の山場
  4. 敗者は点差レベルの罰くじを引いて即実行 → 次のラウンドへ
  5. 先に5勝で決着 → 総合敗者にデカ罰：ケツバット3連発＋視聴者投票で追加1品

星取表は常時掲示（途中参加でも「いま3-2で王手」が一目で分かる）。曲の使い回しは禁止。長引いたら6ラウンド目以降はワンハーフ歌唱に切替、早く決着したら視聴者リクエスト歌唱のエキシビションで締めます。

点差＝罰の重さ（3段階）

点差レベルくじ箱
0.999以下僅差 → 軽罰軽罰くじ
1.000〜2.999中罰中罰くじ
3.000以上大差 → 激罰激罰くじ

点数が読み上げられる瞬間に「激罰キター！」まで決まる仕組み。閾値は初回の実測点差でチューニングします。

罰くじの中身（たたき台・ひろまるさんと一緒に完成させたい）

  1. 軽罰顔にシール追加（終了まで剥がせない）／次ラウンド語尾縛り／変顔札を下げて歌う／ひとことモノマネ
  2. 中罰ケツバット1発／被り物・変装で次の曲／次ラウンドは相手選曲／全力エア熱唱
  3. 激罰ケツバット2連発／レモン丸かじり系／氷水に手30秒／次の曲をキー+4で歌う／変顔写真をサムネに使われる権

視聴者参加の4本柱

1. お題決定権ギフト

各ラウンド冒頭、指定ギフトを最初に投げた人が選曲テーマを決められる（90年代・アニソン・夏うた等のテーマ指定が基本）。ギフトが無いラウンドは無料でルーレット回転。

2. 妨害ギフト

指定ギフトで妨害カード発動（キー±2変更・エコーOFF・被り物・変顔札など）。どちらを妨害するかは視聴者が指定。1曲につき最大2回まで＝点数勝負は壊さない。

3. 応援加点ギフト

各歌唱直前の「応援タイム30秒」のギフト量に応じて +0.2〜最大+1.0点を加点。僅差はひっくり返るが、大差は覆らない上限設定。

4. ピッタリ賞（点数予想で賞金・参加無料）

歌唱前に「86.534」形式で点数をフル予想してコメント（歌い出しで締切・1人1回）。参加は完全無料、各賞先着1名。

一致例（正解 86.534 の場合）賞金
下3桁一致◯◯.5341,000円
下4桁一致◯6.5345,000円
完全一致86.53410,000円

支払いはAmazonギフトコード等をDMで送付。1配信の賞金総額は上限3万円（到達したらその回は終了）。

ひろまるさんへのお願い

ひろまるさん

  1. 8/7（金）前後の都合を共有（上の「まずこれだけ」）
  2. 罰くじの中身を一緒にブラッシュアップ（足す・削る自由）
  3. 費用とギフト収益の分配ルールを事前にすり合わせ
  4. 当日は喉と尻を仕上げてくる（最大18曲＋ケツバットがあります）

この企画の体制

  1. たろ vs ひろまるの2人企画（むらさこさんの担当役はなし。観戦・参戦アイデア歓迎）
  2. ピッタリ賞の判定は2人で兼任（締切時にコメント欄をスクショ→結果発表時に照合）

全員で守ること

  1. ピッタリ賞は参加無料を徹底（ギフトを参加条件にしない。「コメントだけで誰でも参加無料」を毎回明言）
  2. ケツバットはおもちゃバット＋軽く当ててリアクションで盛るバラエティ演出に統一（ガチの痛がり方はNG）
  3. 罰は部屋を汚さない・機材を濡らさないものに限定（食べ物系は店に持ち込み可否を事前確認）
  4. お店の配信許可の条件に必ず従う
  5. 入店後の音量リハを必ずやる（マイクとミュージックのバランス）

懺悔会

あなたの懺悔、たろが裁きます。有罪か、無罪か。

日程：未定（8月以降で調整中） 場所：自宅配信 尺：約3時間 生配信
おしらせ

この企画はたろ1人で回すソロ企画です

ひろまるさん・むらさこさんの出番はありませんが、「こういう懺悔ネタ面白そう」「進行こうしたら？」のアイデアは大歓迎。日程はかえれま10・カラオケ5番勝負との並びを見て決めます。

企画概要

  • 視聴者が「🙏」とコメント → たろが承認して小窓（ゲスト枠）に招待 → 懺悔してもらう
  • たろが深掘りで詳細を引き出し、最後に「有罪」か「無罪」の一言ジャッジ——毎回オチが付く構造
  • 1人3分目安の完全当日制。1本目はたろ自身の懺悔からスタート（温度感の見本）
  • 懺悔の申請が途切れたら無理に引っ張らず雑談配信に切替（ゆるく回す）

流れ（1人あたり約3分）

  1. 「🙏」とコメントした人の中から、たろが承認して小窓へ
  2. 上げる直前に口頭で注意事項を確認
  3. 懺悔を披露
  4. たろの深掘りタイム（ここが本編）
  5. 判定：有罪 or 無罪 → 次の人へ

参加ルール（配信の固定コメントに掲示）

  • 懺悔したい人は「🙏」とコメント → 小窓にご招待
  • 本名・店名・会社名は出さない（イニシャルもNG寄りで）
  • 法に触れる話、過激すぎる話はナシ（出たら途中でも終了）
  • 1人3分くらい。笑える懺悔、大歓迎

守ること（配信を止めないための線引き）

  1. 個人・店・会社が特定される話はしない・させない
  2. 法に触れる話・過激すぎる話が出たら、その場で終了して小窓から下ろす
  3. 未成年らしき人は小窓に上げない
  4. 迷ったらTikTokガイドライン優先で安全側に倒す

どんな企画？

  • その町で頑張っている若者・お店・企業を応援し、地元グルメと町の魅力を紹介する
  • 締めのくじで次に行く町が決まる（エリア番号を視聴者がランダムに選ぶ）→ 出た町のおすすめをその場で聞く
  • 地域を盛り上げることは、たろの起業テーマ（地域再興）そのもの。本気でやる企画

目的（5つ）

  1. その町で頑張っている若者・店舗・企業の応援
  2. 地元のグルメ紹介
  3. 地元の魅力紹介
  4. 紹介を通じた企業とのタイアップなど、仕事への連携（ひろまるさん・たろの仕事につながる）
  5. シリーズ化による視聴者数の増加

ルール（5つ）

  1. 最低限の下調べをして行く
  2. 一度行った地域には行かない（全エリアを回るため）
  3. ギフトは「移動費・現地の食費・応援に使う資金」として、使い道を公表して扱う
  4. 動画の最後に、次に行く町をくじ（エリア番号）で視聴者にランダムに選んでもらい、出た町のおすすめを聞く
  5. 視聴者はスポンサーではなく協力者・参加者という位置付け

エリアの分け方（全72）

  • 1〜7：北九州市の7区（門司・小倉北・小倉南・若松・八幡東・八幡西・戸畑）
  • 8〜14：福岡市の7区（東・博多・中央・南・西・城南・早良）
